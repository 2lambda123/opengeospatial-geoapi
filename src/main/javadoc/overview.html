Java interfaces for OGC standards, conformance tests and examples. The normative part is the org.opengis.geoapi module. Implementations can use the org.opengis.geoapi.conformance module for testing themselves.

Links

  • Open Geospatial Consortium
  • GeoAPI home page

Requirements

  • Java 11 or later.
  • JSR-385 units of measurement API.
