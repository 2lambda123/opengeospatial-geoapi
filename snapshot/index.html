GeoAPI SNAPSHOT

The 3.1-SNAPSHOT and 4.0-SNAPSHOT are the unreleased versions of GeoAPI currently in development. Developers working on projects who need long term stability should use the released 3.0.1 version of the API rather than the snapshots, formalized with the OGC 09-083 specification document. The 3.1-SNAPSHOT version contains the same interfaces than 4.0-SNAPSHOT without the incompatible changes.
