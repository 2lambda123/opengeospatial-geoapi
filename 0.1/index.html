GeoAPI 0.1 (archived)

Note: GeoAPI 0.1 is a legacy milestone. See home page for latest release.

Release 0.1 of GeoAPI contains the interfaces attached to early drafts of the specification document OGC 01-009 of the Open Geospatial Consortium OpenGIS Implementation Specification: Coordinate Transformation Services but dropped from the final version of the standard. This release required a Java environment J2SE 1.3 or higher.
