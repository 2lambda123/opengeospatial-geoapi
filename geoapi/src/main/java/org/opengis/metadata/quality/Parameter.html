DQM_Parameter replacement

The DQM_Parameter class defined in ISO 19157 is omitted for unifying parameter handling with the org.opengis.parameter package. The following table maps DQM_Parameter properties to their replacements.

Mapping from DQM_Parameter properties to ParameterDescriptor properties
DQM_Parameter ParameterDescriptor Old type New type
name getName() InternationalString Identifier
definition getName().getDescription() InternationalString InternationalString
description getDescription() Description InternationalString
valueType getValueClass() RecordType Class<?>
valueStructure See replacement listed below

The Description type had only two properties:

  • textDescription which is mapped to getDescription() (same type),
  • extendedDescription (a BrowseGraphic) which currently has no replacement.

See GitHub issue #74: Harmonize ISO 19157 parameter class.

DQM_ValueStructure replacement

The DQM_ValueStructure code list defined in ISO 19157 is omitted in ParameterDescriptor because in an object-oriented language, we rather expect this information to be provided by the object type. The following table maps DQM_ValueStructure code list values to Java or GeoAPI types.

Mapping from code list values to Java types
Code list Java or GeoAPI type Description
bag java.util.Collection Finite, unordered collection of related items that may be repeated.
set java.util.Set Unordered collection of related items with no repetition.
sequence java.util.List Finite, ordered collection of related items that may be repeated.
table java.util.Map An arrangement of data in which each item is identified by means of keys.
matrix org.opengis.referencing.operation.Matrix Rectangular array of numbers.
coverage org.opengis.coverage.Coverage Function to return values for any direct position within its domain.
