GeoAPI 1.0 (archived)

Note: GeoAPI 1.0 is a legacy milestone. See home page for latest release.

Release 1.0 of GeoAPI contained the interfaces developed by the GeoAPI project and presented with the first draft of the specification document OGC 03-064 of the Open Geospatial Consortium GO-1 Application Objects for evaluation along with the draft standard. This release required a Java environment J2SE 1.4 or higher.
