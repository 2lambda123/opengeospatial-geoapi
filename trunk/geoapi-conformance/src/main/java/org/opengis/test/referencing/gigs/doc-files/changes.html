Changes compared to GIGS files

GIGS guidance notice typically requires an accuracy of 10 decimal figures. However some data declared in the original Excel files do not allow such precision. This page documents the necessary changes applied in order to meet the required accuracy. All new values listed below come from the EPSG database version 7.9.
