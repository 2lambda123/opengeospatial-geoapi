Java interfaces from Open Geospatial Consortium specifications. The interfaces were created from UML diagrams found in the ISO and OGC specifications.

Specifications used

  • GO-1 Application Objects Report
  • Feature Geometry (Topic 1)
  • Spatial Referencing by Coordinates (Topic 2) completed by Coordinate Transformation Services
  • Metadata (Topic 11)
  • Grid Coverages Implementation
