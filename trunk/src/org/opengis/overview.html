OpenGIS® interfaces for Java. Interfaces were created from UML diagrams found in the following specifications:
  • Spatial Referencing by Coordinates (Topic 2)
  • Feature Geometry (Topic 1)
  • Grid Coverages Implementation
  • GO-1 Application Objects Report

Interfaces are created using a fixed set of UML to Java mapping. Substantial modifications are declared in the list of changes from UML diagrams. The documentation uses a fixed set of symbols, terms and definitions.

 

Requirements

  • For execution, J2SE 1.3 or later is sufficient. For compilation, J2SE 1.5 or later is required. This is because the specification uses covariant return types in method overriding. Such covariance is allowed at the virtual machine level, but not at the compiler level prior to 1.5.
  • A JSR-108 units implementation.
  • Java Advanced Imaging and Java 3D are recommended at compilation time in order to resolve Javadoc links, but are not required at runtime.
