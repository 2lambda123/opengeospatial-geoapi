Changes from GeoAPI 1.0 to GeoAPI 1.1

  • In CRSFactory.createImageCRS, changed the cs argument type from CoordinateSystem to ObliqueCartesianCS.
  • In Envelope, added convenience methods getMinimum, getMaximum, getCenter and getLength.
  • In SampleDimension, added a Locale argument for methods getDescription and getCategoryNames.
  • In Coverage, added a Locale argument for method getDimensionNames.
  • In Coverage, changed the return type from double[] to float[] for the method evaluate(DirectPosition, float[]).
  • In Coverage, merged getNumSources() and getSource(int) methods into a single getSources() method.
    • float[] for the method evaluate(DirectPosition, float[]).
  • Javadoc fixes.
