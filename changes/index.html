History

List of releases or milestones:

  • GeoAPI 3.0 - Formal release, standardized by OGC 09-083, GeoAPI specification.
  • GeoAPI 2.3 - Unofficial milestones to GeoAPI 3.0.
  • GeoAPI 2.2 - Unofficial milestone.
  • GeoAPI 2.1 - Unofficial milestone.
  • GeoAPI 2.0 - Formal release, standardized by OGC 03-064, GO-1 Application Objects.
  • GeoAPI 1.0 - Unofficial milestone, released with first draft of OGC 03-064.
  • GeoAPI 0.1 - First release, based on the interfaces defined in OGC 01-009 Coordinate Transformation Services.
